Get a List of All Teams

Retrieve a list of all Formula 1 teams.

Request

HTTP Method: GET

Endpoint: /teams

Request Parameters:

GET /teams

Create a New Team

Create a new Formula 1 team.

Request

HTTP Method: POST

Endpoint: /teams

Request Body: JSON object containing team details.


      {
          "TeamID": 3,
          "TeamName": "New Team",
          "Country": "New Country",
          "YearFounded": 2000,
          "ChampionshipsWon": 0,
          "Drivers": ["New Driver 1", "New Driver 2"]
      }

Example:


      POST /teams
      Request Body:
      {
          "TeamID": 3,
          "TeamName": "New Team",
          "Country": "New Country",
          "YearFounded": 2000,
          "ChampionshipsWon": 0,
          "Drivers": ["New Driver 1", "New Driver 2"]
      }

Response

Status Code: 201 Created

Response Body: The newly created Formula 1 team object.


      {
          "TeamID": 3,
          "TeamName": "New Team",
          "Country": "New Country",
          "YearFounded": 2000,
          "ChampionshipsWon": 0,
          "Drivers": ["New Driver 1", "New Driver 2"]
      }

Update Team Driver Details

Update the details of a driver in Formula 1 team.

Request

HTTP Method: PUT

Endpoint: /teams/{team_id}

Request Parameters:

  • team_id (int): The ID of the team to update.

Request Body: JSON object containing updated team details.


   [  
      "Driver X", "Driver Y"
    ]

Example:


    PUT /teams/1
    Request Body:
    [  
    "Driver X", "Driver Y"
  ]

Response

Status Code: 200 OK

Response Body: The updated Formula 1 team object.


    {
        "TeamID": 1,
        "TeamName": "Updated Team A",
        "Country": "Updated Country A",
        "YearFounded": 1995,
        "ChampionshipsWon": 6,
        "Drivers": ["Driver X", "Driver Y"]
    }

Delete a Team

Delete a Formula 1 team by its ID.

Request

HTTP Method: DELETE

Endpoint: /teams/{team_id}

Request Parameters:

  • team_id (int): The ID of the team to delete.

Example:


    DELETE /teams/3

Response

Status Code: 204 No Content

Response Body: None
